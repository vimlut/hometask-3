# Hometask 3

## Endpoins
| **Method** | **Url** | **Action** |
| --- | --- | --- |
| GET | api/notes | get all notes |
| GET | api/notes/:id | get note by id |
| GET | api/notes/stats | get notes stats |
| POST | api/notes | add new post |
| PATCH | api/notes/:id | update note |
| DELETE | api/notes/:id | remove note |
